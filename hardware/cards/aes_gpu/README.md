# Aesthedes C100/065

This card produces an RGB video signal (sync on green).

## Slot 508

Unknown serial number (I forgot to write it down), but it can be distinguished the the presences of the vertical RAM banks.

Connections:

	A: to 611A
	B: to 513C
	C: to 613B
	D, E, F: to Color Current R, G, B

## Slot 513

SN: 25

Connections:

	A: unplugged
	B: to Color Combi Sync out?
	C: to 508B
	D, E, F: to Color Window R, G, B
